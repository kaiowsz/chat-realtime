## :vulcan_salute: Getting started

#### Cloning the repository

```bash
$ git clone https://github.com/kaiowsz/chat-realtime.git

$ cd chat-realtime
```
#

#### Installing dependencies

```bash
$ yarn
```
or
```bash
$ npm install
```

#

#### Running

```bash
$ npm dev
```
or
```bash
$ yarn dev
```
